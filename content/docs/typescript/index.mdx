TypeScript is a statically typed superset of JavaScript that adds optional type annotations and type inference to the language. It supports all JavaScript features while introducing a rich type system, interfaces, and advanced tools for structuring code. TypeScript code is transpiled into plain JavaScript, making it compatible with any environment where JavaScript runs.
