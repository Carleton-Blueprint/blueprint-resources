## Configuration

We recommend starting with the basic configuration or our recommended configuration and adding formatting
options as needed/desired.

Here is our recommended configuration:

```json
{
  "semi": true,
  "singleQuote": true,
  "trailingComma": "all",
  "tabWidth": 2,
  "useTabs": false,
  "printWidth": 120,
  "arrowParens": "avoid",
  "jsxSingleQuote": false,
  "bracketSpacing": true,
  "endOfLine": "lf"
}
```

## IDE/Editor Setup

### VSCode

We highly recommend you turn on "format on save" on VSCode. What this will do is format the file you
are working on when you save effectively turning your save key into a format key.

1. Start by installing the Prettier extension on VSCode. You can search this extension ID in the Extensions tab on the sidebar menu: `esbenp.prettier-vscode`
   to find the Prettier extension or just search for it by name.
2. Next, open VSCode Settings, search for "Default Formatter", and
   select the "Prettier - Code Formatter" option.
3. Finally you can search for the "Format on Save" setting and make sure it is selected.
